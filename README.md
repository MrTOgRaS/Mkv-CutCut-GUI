<a name="english"></a>
## 🇺🇸 English

**MKV Cutcut GUI** is a lightweight, modern GUI front-end for `mkvmerge` (MKVToolNix). It allows you to cut or split MKV files within seconds **without re-encoding**, ensuring zero quality loss.

### ✨ Features
- **Persistent Settings (New):** Automatically remembers your `mkvmerge.exe` path for every startup.
- **Zero Quality Loss:** No re-encoding, just pure splitting.
- **Flexible Modes:** Cut by Time (`HH:MM:SS.mmm`) or by Frame range.
- **Quick Sample:** Automatically extract a custom preview from any part of the video.
- **Track Info:** View detailed track and codec information with one click.
- **Modern Dashboard:** Slate & Cyan themed dark mode for the best user experience.

### 🛠️ Installation & Usage
1. **Download EXE:** Go to [Releases](../../releases) and download `MKV-Cutcut-GUI_v1.1.exe`.
2. **Run from Source:** Ensure [MKVToolNix](https://mkvtoolnix.download/) is installed and run:
```bash
python MkvCutCutGui.py
```

## 🇹🇷 Türkçe

**MKV Cutcut GUI**, `mkvmerge` (MKVToolNix) altyapısını kullanan, modern ve hafif bir kesme aracıdır. Videolarınızı **yeniden kodlama (re-encoding) yapmadan**, saniyeler içinde ve kalite kaybı yaşatmadan kesmenizi sağlar.

### ✨ Özellikler
- **Kalıcı Ayarlar (Yeni):** Programı her açtığınızda `mkvmerge.exe` yolunu tekrar seçmenize gerek kalmaz, otomatik hatırlar.
- **Sıfır Kalite Kaybı:** Yeniden işleme yok, sadece kayıpsız bölme.
- **Esnek Modlar:** Zaman (`SS:DD:SN.mmm`) veya Kare (Frame) numarasına göre kesim.
- **Hızlı Örnek Alıcı:** Videonun ortasından veya istediğiniz yerden otomatik önizleme parçası oluşturma.
- **Video Bilgisi:** Tek tıkla kanal ve codec detaylarını görüntüleme.
- **Modern Arayüz:** Slate & Cyan temalı koyu mod tasarımı.

### 🛠️ Kurulum ve Kullanım
1. **EXE İndir:** [Releases](../../releases) sayfasından güncel `MKV-Cutcut-GUI_v1.1.exe` dosyasını indirin.
2. **Koddan Çalıştır:** Bilgisayarınızda [MKVToolNix](https://mkvtoolnix.download/) kurulu olmalıdır:
```bash
python MkvCutCutGui.py
```

<a name="deutsch"></a>
## 🇩🇪 Deutsch

**MKV Cutcut GUI** ist ein leichtgewichtiges GUI-Frontend für `mkvmerge` (MKVToolNix). Es ermöglicht das Zuschneiden von MKV-Dateien in Sekunden **ohne Neucodierung**, wodurch die Originalqualität erhalten bleibt.

### ✨ Funktionen
- **Permanente Einstellungen (Neu):** Speichert den Pfad zu `mkvmerge.exe` automatisch für jeden Programmstart.
- **Kein Qualitätsverlust:** Rein verlustfreies Splitten ohne Re-Encoding.
- **Flexible Modi:** Schneiden nach Zeit (`HH:MM:SS.mmm`) oder nach Frame-Bereich.
- **Schnellvorschau:** Erstellt automatisch eine Vorschau aus einem beliebigen Teil des Videos.
- **Spur-Infos:** Detaillierte Codec- und Spur-Informationen mit einem Klick anzeigen.
- **Modernes Dashboard:** Benutzerfreundliches Dark-Mode Design in Slate & Cyan.

### 🛠️ Installation & Nutzung
1. **EXE Herunterladen:** Unter [Releases](../../releases) die `MKV-Cutcut-GUI_v1.1.exe` laden.
2. **Vom Quellcode starten:** [MKVToolNix](https://mkvtoolnix.download/) muss installiert sein:
```bash
python MkvCutCutGui.py
```
